# Security Policy

## Supported Versions

The latest minor release line of `@seatmaps.com/angular-lib` receives
security updates. Older lines may receive critical fixes at the maintainers'
discretion.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

**Please do not file public GitHub issues for security vulnerabilities.**

If you believe you have found a security vulnerability in this project, report
it privately through one of the following channels:

- **GitHub Security Advisories (preferred)** — open a private report at
  <https://github.com/Kwiket/jets-seatmap-angular-lib/security/advisories/new>.
- **Email** — send details to `security@kwiket.com`.

Please include as much of the following as possible:

- A description of the vulnerability and its impact.
- Steps to reproduce, including a minimal proof-of-concept if applicable.
- The affected version(s) of the library.
- Any known mitigations or workarounds.

### Response Process

- We acknowledge new reports within **3 business days**.
- We aim to provide an initial assessment within **10 business days**.
- Once a fix is ready, we coordinate disclosure with the reporter and publish
  a GitHub Security Advisory with the patched release.

Thank you for helping keep the project and its users safe.
